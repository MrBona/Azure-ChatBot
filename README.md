# Azure-ChatBot
Creating Azure ChatBot
#### The Major Steps
1. Create Language Resource
2. Create QnA
3. Create Knowladge Base(kb)
4. Create Web App Bot
5. Create Facebook Mesenger App
6. Connect FB Mesenger App with the Web App Bot
7. Test the ChatBot
